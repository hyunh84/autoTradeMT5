==============================================================================================================================
작업 진행 순서
==============================================================================================================================

| 단계  | ------ 목표 ------ | -- 주요 기술 -- | -------- 결과 -------- |
| ----- | ------------------ | --- 0000 ------ | -------- 00 ---------- |
| 1단계 | PyQt GUI 기본 구성 | PyQt5           | 버튼 + 로그창 표시     |
| 2단계 | MT5 연동           | MetaTrader5     | 시세/계좌 출력         |
| 3단계 | 전략 구현          | Python 알고리즘 | 자동 주문 실행         |
| 4단계 | 멀티스레딩         | QThread         | UI 멈춤 없음           |
| 5단계 | 고급 기능          | OpenAI, CSV     | 전략 요약, 백테스트 등 |



1단계. 자동매매 로직 연결
"자동매매 시작" 버튼을 눌렀을 때 실제 매매 알고리즘을 실행

예: 특정 조건에서 Buy/Sell 주문 넣기

✅ 예시: MACD, RSI, Bollinger, Stoch 전략 적용

📈 2단계. 실시간 시세 조회 & 갱신
실시간 가격(틱, 현재가 등)을 1초마다 자동으로 갱신

시세가 바뀌면 UI에 반영 (ex. "현재가: 1020.25")

💸 3단계. 포지션 정보 / 수익률 표시
현재 진입한 포지션 수량, 수익률 표시

"현재 포지션: 롱 0.001 BTC @ 42000 (익절 +2.5%)"

📋 4단계. 주문내역 / 거래로그 테이블
체결된 거래내역을 테이블로 관리

"2025.06.19 10:32:15 | Buy 0.01 @ 42800 | 수수료: 0.2" 형식
